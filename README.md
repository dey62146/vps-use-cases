## 搬瓦工基础操作：买完怎么上手

VPS 买了之后，基本流程就这几步：

1. **收邮件拿 IP 和 SSH 端口**：购买成功后搬瓦工会发邮件，里面有 IP 地址、SSH 端口、初始密码
2. **登录 KiwiVM 面板**：进到后台可以看到服务器状态、流量用量、机房迁移入口
3. **SSH 连接服务器**：Windows 用 PuTTY 或 Xshell，Mac/Linux 直接终端，`ssh root@你的IP -p 端口号`
4. **改 root 密码**：KiwiVM 面板里有"Root password modification"，第一次进去先改掉默认密码
5. **选操作系统**：默认是 Debian 或 CentOS，也可以在面板里一键重装成 Ubuntu 26.04 或其他版本
6. **安装宝塔面板或直接操作**：新手建站推荐先装宝塔，图形化操作比纯命令行友好
